# 证书

## 新增证书

打开 **控制面板** -> **安全性** -> **证书**

![1721890353286](image/certficate/1721890353286.png)

点击 **新增**，打开 **创建证书** 面板

![1721890369541](image/certficate/1721890369541.png)

点击 **下一步**

![1721890416094](image/certficate/1721890416094.png)

选择 **创建自我签署证书**，点击 **下一步**

后面的依次填写信息创建即可。

Mysql容器的安装参考: [MySQL 安装](synology.md#mysql-install)
