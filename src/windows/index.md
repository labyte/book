# Windows

## 安装

**官网下载**

[官网](https://www.microsoft.com/zh-cn/software-download/windows10) 可以更新当前系统，以及通过下载工具下载系统制作成 U盘 或者 下载IOS文件


当下载工具后运行，可选择(win10和win11相同操作):

- **U盘**：制作U盘
- **IOS文件**：保存文件
